# CY-Connect

CY-Connect est un jeu en langage C inspiré du Puissance 4.

Le but du jeu est d’aligner **5 pièces identiques** sur un plateau de **8 colonnes** et **6 lignes**.

Le projet est réalisé en version **modulaire**, c’est-à-dire que le code est séparé en plusieurs fichiers pour être plus clair, plus propre et plus facile à modifier.

---

## Compilation

Pour compiler le projet, ouvrez un terminal dans le dossier du projet, puis tapez :

```bash
make
